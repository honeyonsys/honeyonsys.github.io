honeyonsys google map

Honeyonsys google map is a wordpress plugin to place google map with address marker on the post/page/template. You can place multiple numbers of markers in a single map.

Documentation

Installation

1.Download the plugin zip folder. You can download the latest plugin zip from author's github repository honeonsys google map.

2.After download the zip, place that into the wp-content/plugins folder of your wordpress site and extract it.

3.Now go the wp-admin -> Plugins and activate the honeyonsys google map plugin.
